Conceitos da Tag HEAD

Pergunta 1:

A tag HEAD é utilizada para fornecer informações sobre o documento e conter elementos como <meta>, <title>, <base>, entre outros.

Pergunta 2:

A tag <meta> é utilizada para fornecer metadados, <title> define o título da página, e <base> especifica a URL base para todas as URLs relativas na página.

Exemplo:
<meta name="description" content="Descrição da página">
<title>Minha Página</title>
<base href="https://www.exemplo.com/">

Pergunta 3:

Charset refere-se ao conjunto de caracteres utilizado para codificar o texto. Sua utilidade é garantir que os caracteres sejam interpretados corretamente.

Pergunta 4:

Se o charset não for definido, o padrão geralmente utilizado é o UTF-8.

Pergunta 5:

UTF-8: Padrão amplamente utilizado, suporta diversos caracteres incluindo internacionais.

ASCII: Conjunto básico de caracteres, utiliza 7 bits para representar cada caractere.

ANSI: Conjunto de caracteres estendido baseado no ASCII, utilizado principalmente no Windows.

ISO-8859-1: Conjunto de caracteres latino-1, utilizado para idiomas europeus.

Pergunta 6:

Para usar símbolos e emojis, pode-se utilizar caracteres Unicode diretamente no HTML.

Exemplo:
<p>Olá! 😊</p>
